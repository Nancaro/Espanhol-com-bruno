# Espanhol com Bruno

Sitio web oficial de **Espanhol com Bruno**, proyecto educativo enfocado en la enseñanza de español para brasileños con clases online, metodología comunicativa y acompañamiento personalizado.

## Lo que representa este proyecto

Este sitio no solo vende clases: comunica una propuesta educativa con identidad propia.

- Enseñanza de español con foco práctico (viajes, trabajo, exámenes y vida real).
- Relación cercana entre profesor y alumno.
- Acceso claro a planes, materiales y contacto.
- Confianza a través de prueba social, credenciales y estructura profesional.
- Impacto social mediante campañas solidarias (por ejemplo, alianzas con ONG y beneficios para donadores).

## Sobre el sitio web

Es una landing page estática orientada a conversión, construida para:

- Presentar al profesor y su método.
- Mostrar planes, cursos y productos digitales.
- Resolver dudas frecuentes.
- Llevar al usuario a WhatsApp o formulario de agendamiento.

## Stack y arquitectura

- `HTML5` + `CSS3` + `JavaScript` vanilla (sin framework).
- Estructura principal en un solo archivo: `index.html`.
- Archivos auxiliares SEO/infra:
  - `robots.txt`
  - `sitemap.xml`
  - `CNAME` (dominio personalizado)
- Assets locales:
  - `2-removebg-preview (1).png`
  - `Red Gradient Profile Photo Instagram Post (1).png`

## Ejecución local

Al ser un sitio estático, puedes abrir `index.html` directamente en el navegador o usar un servidor local simple.

Ejemplo con VS Code:

1. Abrir la carpeta del proyecto.
2. Ejecutar la extensión **Live Server** sobre `index.html`.

## Objetivo de negocio

Convertir visitantes en alumnos mediante una experiencia clara, confiable y emocional:

- Mensaje de valor fuerte en la primera pantalla.
- CTA directos a WhatsApp y formulario.
- Secciones diseñadas para reducir objeciones (FAQ, resultados, método, planes).

## Mantenimiento recomendado

- Mantener precios, enlaces y CTAs actualizados.
- Revisar periódicamente textos de campañas y popups.
- Optimizar imágenes para mejorar carga.
- Validar `sitemap.xml` y `robots.txt` cuando se agreguen nuevas páginas.
